# 📊 PhonePe Transaction Analysis Dashboard

A comprehensive **Power BI Dashboard** that analyzes PhonePe digital payment transactions across India. This project transforms raw transaction data into interactive visualizations, helping users explore transaction trends, payment categories, and regional performance.

---

## 📌 Project Overview

The objective of this project is to analyze PhonePe transaction data and present meaningful insights through an interactive Power BI dashboard. The dashboard enables users to monitor transaction volume, transaction value, and payment trends across different states and time periods.

---

## 🚀 Features

* Interactive Power BI dashboard
* State-wise transaction analysis
* District-wise transaction insights
* Transaction volume and transaction value analysis
* Year-wise and quarter-wise trend analysis
* Category-wise payment distribution
* Dynamic filters and slicers
* Clean and user-friendly interface

---

## 🛠️ Tools & Technologies

* Power BI
* Power Query
* DAX
* Microsoft Excel
* Data Modeling

---

## 📊 Dashboard Insights

* Analyze transaction trends across India.
* Compare transaction values over different years and quarters.
* Identify top-performing states and districts.
* Explore category-wise payment distribution.
* Monitor KPIs through interactive visualizations.
